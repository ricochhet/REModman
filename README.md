# REModman
RE Engine Mod Manager

## Building

Download the following tools if you do not have them already installed.
- .NET 7.0 [download here](https://dotnet.microsoft.com/en-us/download/dotnet/7.0)
- Run the `build.bat` or `build-publish`.

## Supported Games
- Monster Hunter Rise
